# Changesets

Hello! This folder contains changeset files for version management.

## What are changesets?

Changesets are a way to manage versions and changelogs with a focus on mono-repos and multi-package repositories.

## How to create a changeset

```bash
npm run changeset
```

Follow the prompts to describe your changes.
